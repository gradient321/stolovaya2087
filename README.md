Данный проект является программной моделью системы учета заказов в школьной столовой. Он разработан для упрощения процесса выбора и заказа блюд
Функционал
Система предусматривает два основных режима:

Режим ученика:

Регистрация и авторизация.

Просмотр актуального меню на неделю.

Пополнение личного счета (условно).

Выбор блюд из меню и оформление заказа.

Просмотр истории своих заказов и состояния счета.

Режим администратора (и повара):

Добавление и редактирование блюд в меню.

Установка цен.

Просмотр всех заказов за день.
